
Create your catkin workspace and pull soma-skins-franka repo as a new package into catkin_ws/src folder


1. Running Kinesthetic control:

    --> In a terminal run : roslaunch rosbridge_server rosbridge_websocket.launch
    --> In another terminal run : rosrun soma-skins-franka kinesthetic_control.py


2. Running keyboard control:

    Installation:
        --> Download telleop_twist_keyboard from https://github.com/ros-teleop/teleop_twist_keyboard
        --> Put it inside catkin_ws/src

    Running:
        --> In a terminal run : roslaunch rosbridge_server rosbridge_websocket.launch
        --> In another terminal run : rosrun soma-skins-franka keyboard_control.py
        --> In another terminal run : rosrun teleop_twist_keyboard teleop_twist_keyboard.py _speed:=0.05 _turn:=0.8
